What are the Job Titles for your group's ideal jobs? How do each of these rank in terms of demand from employers?

The job titles for our teams ideal jobs are Database administrator, Cyber security architect, IT manager/coordinator and Network architect. Each of these jobs have a variety of different skill sets therefore it is hard to rank these jobs, however according to CIO.com the highest job in demand is a security professional (Cybersecurity architect and IT Manager) then database administrator and then lastly a network architect/administrator. Despite the rankings of these jobs, these professions are all in a high demand and still pay well making each of these ideal jobs a great goal to try and achieve in the future.

How do the IT-specific skills in your required skill set rank in terms of demand from employers?

The IT-specific skill set of our group does meet certain demands from employers such as programming language which is considered essential in the IT industry. Programming languages and management are the most popular skills to have. Aside from programming languages there are a lot of skills related to a high sophisticated understanding of maintaining various systems and architectures such as networks, firewalls, wireless systems and desktops that can only be obtained through years of real work experience in the IT world. What are the three highest ranked IT-specific skills which are not in your required skill set?

JAVA, Microsoft Windows, project management and SAP which are all software related fields not found in our group’s ideal skill sets. JAVA is a highly sought after IT skill that is useful in most operating systems as JAVA runs on its own virtual machine in order to talk to the hardware. JAVA is written in a high level language and then compiled to run on its own, in comparison to javascript which is an interpretive language which compiles as the program is run. With the ever growing market of computers, especially in the present day landscape where working at home has become a mainstay in office related work. Many of these users will be on the Windows operating systems as the windows market share is around 87% against Mac OS, Linux, Chrome OS and others, meaning that being proficient in windows related skills are crucial. As for SAP, an abbreviation for Systems Applications and Products within Enterprise Resource Planning. It is not only a centralised system that connects all departments of a company together but helps the communication, providing information and reducing redundancies and duplication between these departments. It makes all processes within a company more efficient and productive and that in itself is self explanatory for why skills within SAP is an extremely in demand skill.
